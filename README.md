# 🌍 Travel Page Automation

An automated workflow built in **n8n** that turns a single city name into a fully designed, shareable HTML travel page — generated by AI, backed up to GitHub, and sent for email approval.

---

## ✨ What it does

1. A user types a **city or country name** into a Google Sheet.
2. The workflow asks a free **AI model (Groq)** to return structured travel data as JSON.
3. A **Code node** injects that data into a hand-built HTML template — header, 3 images, attraction cards, and an interactive map.
4. The finished page is **backed up to GitHub** and served live via **GitHub Pages**.
5. A **preview email** is sent with an *Approve* / *Reject* choice.
6. **Approved** → the page link is saved back to the sheet.
   **Rejected** → an optional note is collected and a brand-new version is generated.

---

## 🧭 Flow overview

```
Google Sheets Trigger
        ↓
Set (extract city)
        ↓
Code node  →  build the Groq prompt
        ↓
HTTP Request  →  Groq API (returns JSON)
        ↓
Code node  →  assemble the HTML page
        ↓
GitHub  →  backup every run
        ↓
Gmail  →  send preview, wait for approval
        ↓
   ┌────────────────────┐
   ↓ Approved           ↓ Rejected
Update Sheet      Collect note → regenerate
```

## 🧰 Tech stack

| Tool | Role |
|------|------|
| **Google Sheets** | Trigger + storage of approved page links |
| **Groq** (`llama-3.3-70b-versatile`) | Free AI model — returns structured JSON |
| **n8n Code nodes** | Build the prompt and assemble the HTML |
| **GitHub + GitHub Pages** | Backup of every page + live hosting |
| **Gmail (Send & Wait)** | Email preview with approve / reject |
| **Leaflet + OpenStreetMap** | Interactive map with a pin per attraction |

## 🔑 Setup — getting the API key

This workflow uses **Groq**, which is free and requires no credit card.

1. Sign in at [console.groq.com](https://console.groq.com)
2. Open **API Keys** → **Create API Key** → copy it (shown once)
3. In the n8n workflow, open the **HTTP Request** node (the Groq call)
4. Under **Headers**, set:
   ```
   Authorization: Bearer YOUR_GROQ_API_KEY
   ```

Google Sheets, Gmail, and GitHub each connect through their own OAuth credential inside n8n.

## 📝 Notes & known limitations

- **Images:** fetched from Wikipedia by attraction name — accurate for major landmarks, but niche venues (e.g. small restaurants) may fall back to a generic image.
- **GitHub Pages delay:** a newly pushed page takes 1–3 minutes to go live, so its link may briefly return a 404 right after generation.
- **Email styling:** email clients strip CSS, JS, and iframes, so the email shows an inline-styled preview with a button to the full live page.

---

## 📦 Tech notes

- The AI returns **structured JSON**, not raw HTML — this keeps output reliable and lets the template control the design.
- The HTML page is assembled in a Code node and pushed to GitHub as a base64-encoded file on every run.
